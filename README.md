# My Coding Portfolio Website

This is my portfolio website! It is a small web app built with ReactJS and CSS demonstrating my skills, projects, and journey in the path of development. You can see it for yourself at [edrisadel.dev](https://edrisadel.dev)

<img width ="1280" src="https://github.com/EdrisAdel/PortfolioWebsite/blob/main/src/images/screenshot.png">



## Features

- **Responsive Design:** The portfolio is designed to be accessible and visually appealing across various devices and screen sizes.

- **Skills Section:** A section to showcase my skills and proficiency in different programming languages, frameworks, and tools displayed in varying lengths.

- **Contact Form:** An interactive contact form allowing viewers to reach out to me directly.

## Technologies Used

- ReactJS: A JavaScript library for building user interfaces.
- Web3Forms: An API based form backend that allows for Form to Email contact.
